# 이력서 포트폴리오 — Magic Patterns 제작, Codex로 다듬음

이 프로젝트는 [Magic Patterns](https://magicpatterns.com)에서 생성한 디자인을 기반으로 만든 이력서 포트폴리오이며,
Codex를 통해 코드와 문서 표현을 다듬었습니다. 원본 디자인은 다음 링크에서 확인할 수 있습니다:
[Source Design](https://www.magicpatterns.com/c/vbuguyvujj8ewavywgz4du)

## 시작하기

1. `npm install`
2. `npm run dev`

## 기술 스택

- Vite + React + TypeScript
- Tailwind CSS
